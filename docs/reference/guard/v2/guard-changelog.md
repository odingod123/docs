# Guard Changelog

<LastUpdated/>

- ## 3.0.0

> 更新时间：2022.1.4

1. 添加 Guard 组件
2. 完善 MFA 功能：支持人脸识别、电子邮箱验证以及 TOTP 验证器下载
3. 增加登录安全策略：支持账号锁定 / 图形验证码、首次登录修改密码以及密码轮换策略
4. 完善 CSS 相关功能，支持多租户、登录注册协议配置
5. 全面优化各类登录注册流程逻辑细节，并全新升级登录框 UI
